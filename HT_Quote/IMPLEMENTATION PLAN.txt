KEY BUSINESS RULES
Quotation Numbers: Auto-generated as Q-YYYY-NNN, reset annually

AMC Handling: Optional line items below main products, select from predefined descriptions

Pricing: Services use man-day rates, products use unit pricing

Currency: Base in USD, convert to MVR using exchange rates

Validity: Configurable default (14 days), editable per quotation

T&C: Select multiple templates, auto-suggest based on product types
